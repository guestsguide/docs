# Projektnotizen

## Guests Guide – Produkteinschränkungen

Folgende Features werden aktuell **nicht** unterstützt und dürfen in Texten nicht als verfügbar dargestellt werden:

- **Keine Live-Aktivitäten:** Echtzeit-Daten (z.B. automatische Wartezeiten von Sensoren) werden nicht unterstützt. Informationen können aber regelmäßig manuell über das CMS aktualisiert werden.
- **Keine iframes:** Einbettungen per iframe sind nicht möglich.
- **Keine Push-Notifications:** Da es sich um eine Webapp handelt, können keine Push-Benachrichtigungen verschickt werden.
- **Keine Buchungen im Guide:** Leistungen (Massagen, Tickets, Bestellungen etc.) können nicht direkt im Guide gebucht werden. Stattdessen kann auf externe Buchungssysteme verlinkt werden.
